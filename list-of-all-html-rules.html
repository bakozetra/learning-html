  • If you use :<select > <textarea> <video> the <controls>attribute is present.
  • Closing tag comments should include the id or class of the element
  • You have to follow this rules:correct tag order,lowercase tags,closing tags,quoting attributes
  • Keep track of your code to make sure that there exists a proper tag order.
  • When you link to a page on another website, all you have to do is specify the absolute URL for it. However, when you link to a page on your own website you have to take into consideration the directory that page is located in
  • When you put it in a link, you're saying that the link should start from the root directory of the website.
  • To link to a named anchor, add a # symbol and the name of the anchor to the end of a URL
