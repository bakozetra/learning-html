About the nesting rules

Opening tag and closing tag is nesting 's rules Why is it relevant to remember ?If you create a document and use an html ,you must follow these structur because it help you to become a good at coding. accoding that definition you have to follow these rules:

  • We should use a lowercase for the element name because a lowercase word is a term which describes a collection of HTML elements
  • The element we need to put on element <head> are: TITLE (required) SCRIPT, STYLE CDATA ISINDEX, BASE, META, LINK (empty) OBJECT (see content model below)
  • Define a doctype at the top
  • You can't have end tag without a start tag.
  • Every start tag needs to start with this symbol "<" and ending of ">"
  • Every end tag needs to start with this symbol ""
  • We can't have <ol> as child of <lu> only valid tags to put inside an<ul> is an <li>.
  • we can't put body inside head because head tags don't belong inside of body tags.
  • Element <li> not allowed as child of element body in this context.
  • A document must not include more than one visible main element.
  • The main element must not appear as a descendant of the <nav> element.
  • We can have<a>,<abbr>,<address>,<area> if it is a descendant of a map element)
  • We can use this element <li> Contexts in which this element can be used: Inside <ol>,<ul>, <menu> elements.
